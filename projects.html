📁 项目展示

以下项目主要围绕无人机侦测、无线信号处理与深度学习建模展开。

1. 无人机侦测与威胁等级评估系统

基于多传感器融合（电磁、光电、红外、无线电）实现目标无人机的发现、识别与威胁等级评估。

技术亮点： 信号特征提取、频谱分析、多源信息融合、威胁等级分类模型、自研评估算法。

UAV Detection Multi‑sensor Fusion Threat Assessment

2. 基于深度学习的无线信号智能分类

构建 LSTM + CNN 级联合网络，输入多维时序特征，实现对脉冲及通信信号类型的自动识别。

方法特色： 使用对比学习进行特征预训练，结合监督微调提升小样本场景下的鲁棒性。

CNN · LSTM Contrastive Learning Signal Classification

3. DOA（到达角）估计与定位系统

融合协方差矩阵特征与卷积神经网络，实现复杂电磁环境下的 DOA 估计与目标定位。

相关内容： 与传统 MUSIC / ESPRIT 算法对比，分析在多径、低信噪比条件下的性能表现。

DOA Estimation Array Signal Processing CNN
